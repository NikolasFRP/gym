C. Найди id
Ограничение времени	5 секунд
Ограничение памяти	256Mb
Ввод	стандартный ввод или input.txt
Вывод	стандартный вывод или output.txt
На курсе по алгоритмам в Яндекс.Практикуме зарегистрировано n
студентов. За последние несколько дней сайт посещали все студенты, кроме двоих.
Определите ID этих двух студентов.

Формат ввода
В первой строке записано натуральное число n, 2 ≤ n ≤ 10^3.
Во второй строке записаны n − 2 различных натуральных чисел, не больших n
 —– это ID студентов из журнала посещения.

Формат вывода
Выведите два пропущенных ID в порядке возрастания.
Пример

Ввод
7
6 4 1 2 3

Вывод
5 7
